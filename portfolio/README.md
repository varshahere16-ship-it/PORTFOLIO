# P Versha Portfolio

**Author:** P Versha

## Project Description
A modern, glassmorphism-style portfolio website showcasing projects, skills, experience, education, certifications, and contact details.

## Features
- Responsive layout (mobile-first adjustments)
- Glassmorphism UI design
- Scroll-reveal animations (with reduced-motion support)
- Animated typing effect
- Cursor glow / ambient background effects
- Accessible markup (ARIA labels where appropriate)

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## How to Run the Project
### Option 1: Live Server (Recommended)
1. Install the **Live Server** extension in VS Code.
2. Right-click `portfolio/index.html` → **Open with Live Server**.

### Option 2: Open in Browser
1. Double-click `portfolio/index.html`.
